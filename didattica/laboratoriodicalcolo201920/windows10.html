Utilizzare la Shell di Linux su Windows

Note a cura della Dott. G. D`imperio


Per chi è curioso di capire meglio cosa è Linux su Windows, trovate qualcosa su Wikipedia . Tra le possibili distribuzioni linux consigliamo di scegliere Ubuntu o Debian.

Questa è una guida in inglese che spiega i prerequisiti e i passi da fare per avere una installazione della shell Linux su Windows con interfaccia grafica.

Qui comunque sono estratti in maniera sintetica i passaggi necessari.

  1. Dal Microsoft Store scaricare e installare l’applicazione Ubuntu
  2. Quando aprirete l’applicazione avrete un terminale come questo
    Vi verrà chiesto di creare un nuovo utente UNIX con nome, password e conferma della password (è normale che non si veda nulla sul terminale quando digitate le password).
    User e password potete sceglierli arbitrariamente e sono indipendenti da quelli di Windows, sono relativi alle operazioni che svolgete nella shell Linux.
  3. Per avere l’interfaccia grafica funzionante dovrete installare un X server per Windows come VcXsrv
  4. Una volta installato e avviato VccXsrv digitate sulla shell Linux
    echo "export DISPLAY=localhost:0.0" >> ~/.bashrc
  5. Per fare in modo che i cambiamenti abbiano effetto, chiudete e riaprite Ubuntu.

Installare le applicazioni necessarie per il Laboratorio di Calcolo

Se avete già installato alcune cose attraverso Windows (tipo gcc, emacs, ecc…) non funzioneranno per Ubuntu, ma dovrete reinstallare tutto attraverso la shell di Linux come spiegato sotto.
Prima di tutto scaricate gli aggiornamenti di Ubuntu digitando sulla shell

   sudo apt update
   sudo apt upgrade

Il comando sudo serve a eseguire i comandi come amministratore, ed è necessario per operazioni tipo installare pacchetti o aggiornamenti.

  • Per installare l’editor (si consiglia di usare emacs) digitare sulla shell il comando
   sudo apt-get install emacs
  • Per il compilatore
   sudo apt-get install gcc
  • Per avere python3 con numpy e matplotlib
   sudo apt-get install python3 python3-tk python3-pip3
   pip3 install numpy matplotlib

A questo punto dovreste avere il necessario per svolgere a casa gli esercizi che avete fatto in laboratorio su Windows10!
